# LifeGraph

> Quantify your life like an engineer.

**LifeGraph** is an open-source life tracking system designed for engineers and data-driven people.

Track sleep, steps, mood, and deep work, then generate analytics and insights from your personal data.

Instead of using multiple apps and dashboards, LifeGraph lets you **own your life data** and analyze it locally.

---

# ✨ Features

* 📊 Life metrics tracking (sleep, steps, mood, deep work)
* 📈 Automatic analytics charts
* 🧠 Correlation analysis (e.g., sleep vs mood)
* ⌚ Garmin data sync (optional)
* 🤖 Telegram bot logging
* ⚡ Fast local analytics using Python
* 🔐 Your data stays local

# 🚀 Quick Start

Follow these steps to run LifeGraph locally and generate your first life analytics charts.

---

## 1 Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/lifegraph.git
cd lifegraph
```

---

## 2 Install dependencies

```bash
pip install -r requirements.txt
```

---

## 3 Initialize database

Create the SQLite database:

```bash
python -m app.db
```

This will create:

```
data/life.db
```

---

## 4 Generate demo data

To quickly test the analytics pipeline, generate sample life data:

```bash
python scripts/generate_demo_data.py
```

This will generate about **30 days of example metrics**.

Example data:

| date       | sleep | steps | mood | deepwork |
| ---------- | ----- | ----- | ---- | -------- |
| 2026-03-01 | 7.5   | 8200  | 4    | 3        |
| 2026-03-02 | 6.8   | 6000  | 3    | 2        |
| 2026-03-03 | 8.0   | 10000 | 5    | 4        |

---

## 5 Generate analytics charts

Run:

```bash
python -m analytics.charts
```

This will generate charts:

```
charts_sleep.html
charts_steps.html
sleep_mood.html
```

Open them in your browser.

# 🧠 Why LifeGraph?

Most life tracking apps:

* lock your data
* hide analytics
* limit export

LifeGraph lets you:

* own your data
* analyze everything
* build your own metrics

Think of it as:

```
Prometheus + Grafana
for your personal life
```

But lightweight and local.
